---
## Usage
{% highlight bash %}
resizeBrowser = <<text value of width and height>
{% endhighlight %}

## Examples

{% highlight bash %}
resizeBrowser = ['320','480']
{% endhighlight %}

{% highlight bash %}
resizeBrowser = ['1024','768']
{% endhighlight %}

{% highlight bash %}
resizeBrowser = [width: 320 , height: 480 ]
{% endhighlight %}
