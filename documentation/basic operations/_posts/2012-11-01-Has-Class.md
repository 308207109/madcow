Checks that the element has all of the CSS class names provided.

## Usage

{% highlight bash %}
[MADCOW:htmlElementReference].hasClass = "<<field CSS values>>"
{% endhighlight %}

## Examples

{% highlight bash %}
country.hasClass = "span3"
country.hasClass = floated-right span9
{% endhighlight %}
