## Usage

{% highlight bash %}
[MADCOW:htmlElementReference].hasClass = [MADCOW:field CSS value]
{% endhighlight %}

## Examples

{% highlight bash %}
country.hasClass = "span3"
country.hasClass = floated-right span9
{% endhighlight %}
