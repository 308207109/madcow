title: Store Value

category: basic operations
---

## Usage
{% highlight bash %}
[MADCOW:htmlElementReference.storeValue = <<text value to store>>
{% endhighlight %}

## Examples

{% highlight bash %}
aInputId.storeValue = dummyParam
{% endhighlight %}

{% highlight bash %}
billingCenter_common_searchPolicy_remainingBalance.storeValue = paymentAmount
{% endhighlight %}
