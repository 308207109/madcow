## Usage

Store the parameter:

{% highlight bash %}
[html element reference].store = MyProperty
{% endhighlight %}

Use the stored parameter:

{% highlight bash %}
searchField.value = @MyReferenceNumber
{% endhighlight %}

## Examples

{% highlight bash %}
addressLine1.store = EnteredAddress
verifyText = @EnteredAddress
{% endhighlight %}
