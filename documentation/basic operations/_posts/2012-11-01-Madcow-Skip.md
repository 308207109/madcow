title: Madcow Skip

category: basic operations
---

Used in subsequent tests in the same workbook to skip a step specified in the first test.

## Usage
{% highlight bash %}
madcowSkip
{% endhighlight %}

## Examples
{% highlight bash %}
madcowSkip
{% endhighlight %}
