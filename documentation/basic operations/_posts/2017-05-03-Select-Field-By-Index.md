## Usage
{% highlight bash %}
[MADCOW:htmlSelectReference].selectFieldByIndex = <<index>>
{% endhighlight %}

## Examples
{% highlight bash %}
carCylinders.selectFieldByIndex = 1
{% endhighlight %}
