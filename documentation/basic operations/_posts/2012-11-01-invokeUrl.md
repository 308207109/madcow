## Usage
{% highlight bash %}
invokeUrl = [URL_REFERENCE]
{% endhighlight %}

See the URL [Configuration](Configuration#urlConfiguration) Section for more details on configuring the URL Reference.


## Examples
{% highlight bash %}
invokeUrl = http://projectmadcow.com
invokeUrl = PROJECT_MADCOW_SITE
{% endhighlight %}
