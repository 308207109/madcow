Checks that the element has all of the CSS styles provided.

## Usage

{% highlight bash %}
[MADCOW:htmlElementReference].hasStyle = [MADCOW:field style values]
{% endhighlight %}

## Examples

{% highlight bash %}
country.hasStyle = "display: block;"
country.hasStyle = display: block;line-height: 28px;
{% endhighlight %}
