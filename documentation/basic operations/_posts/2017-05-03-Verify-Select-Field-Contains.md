## Usage

{% highlight bash %}
[MADCOW:htmlElementReference].verifySelectFieldContains = <<[List of values]>>
{% endhighlight %}

## Examples

{% highlight bash %}
countryList.verifySelectFieldContains = ["Australia", "New Zealand", "UK"]
{% endhighlight %}
